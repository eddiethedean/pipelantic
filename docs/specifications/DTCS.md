# DTCS 2.0 (canonical)

The current Data Transformation Contract Standard is published upstream:

**[DTCS SPEC.md](https://github.com/eddiethedean/dtcs/blob/main/SPEC.md)**

Version 2.0.0 adds the Portable Relational Profile and
`dtcs.transform-plan/1`. Installable packages use `dtcs>=0.12`.

## Local archive

A vendored [DTCS 1.0 snapshot](DTCS_SPEC.md) is kept for historical comparison
only. When the archive and the published repository differ, the GitHub
`SPEC.md` is authoritative.
